# Tech Stack

**Date:** 2026-04-27

## Language & Frameworks
- **JavaScript / React**: Core application code.
- **React (v16.10.2)**: Core UI framework.
- **react-scripts (v5.0.1)**: Build system (Create React App).
- **Sass (v1.32.8)**: Styling preprocessor.

## Key Dependencies
- **lottie-react**: For rendering Lottie animations.
- **react-reveal**: For scroll animations.
- **react-headroom**: For hiding/showing header on scroll.
- **colorthief**: For dominant color extraction from images.
- **react-twitter-embed**: For embedding Twitter feeds.
- **gh-pages**: For deployment to GitHub Pages.

## Testing
- **Jest / Enzyme**: Testing framework and React testing utility (`enzyme-adapter-react-16`, `jest-canvas-mock`).

## Configuration
- `.prettierrc` / `.prettierignore`: Code formatting rules.
- `package.json` (eslintConfig): React app defaults.
- `fetch.js`: Pre-build script to fetch dynamic data.
